# Life God Game

This folder owns the full Life God Game easter egg.

- Route entry: `../page.tsx`
- Main React shell: `LifeGodGame.tsx`
- Simulation: `simulation/`
- Rendering: `rendering/`
- Player input: `input/`
- UI overlays and HUD: `ui/`
- Shared types: `types/`

Keep God Game changes inside this folder unless the task explicitly needs to touch the trigger in `components/EasterEggs.tsx` or the route wrapper in `../page.tsx`.

Current trigger: type `codex`.
